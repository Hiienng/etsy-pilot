📘 Mathematical Dictionary

Bernoulli Trial

Bernoulli trial là một phép thử ngẫu nhiên chỉ có hai kết quả khả dĩ: thành công (success) hoặc thất bại (failure).

Notation:

X ∼ Bernoulli(p)

P(X = 1) = p,   P(X = 0) = 1 − p

Examples:

  • Tung đồng xu: ngửa = 1, sấp = 0
  • Email là spam (1) hoặc không spam (0)
  • User click vào quảng cáo hay không
ML Connection:
  • Binary classification labels
  • Logistic Regression giả định target ∼ Bernoulli
  • Cross-entropy loss xuất phát từ Bernoulli likelihood

Binomial Distribution

Binomial distribution mô tả số lần thành công trong n Bernoulli trials độc lập.

Notation:

X ∼ Binomial(n, p)

P(X = x) = C(n, x) px (1 − p)n − x

  • Số mặt ngửa khi tung n đồng xu
  • Số user click trong n lượt hiển thị
ML Connection:
  • Model đánh giá tỉ lệ conversion
  • A/B testing

Random Variable

Random Variable (RV) là một hàm ánh xạ từ không gian mẫu Ω vào tập số thực ℝ.

X : Ω → ℝ

  • X = số email spam trong ngày
  • X = giá nhà
  • X = label của dữ liệu
ML Connection:
  • Feature = random variable
  • Target variable = random variable

IID (Independent and Identically Distributed)

Các biến ngẫu nhiên được gọi là IID nếu:

  • Độc lập với nhau (Independent)
  • Có cùng phân phối (Identically Distributed)

X₁, X₂, ..., Xₙ ∼ P

ML Connection:
  • Hầu hết thuật toán ML giả định dữ liệu IID
  • Violation → data leakage, concept drift

Likelihood

Likelihood đo mức độ "hợp lý" của tham số khi quan sát dữ liệu.

L(θ | x) = P(x | θ)

Với Bernoulli:

L(p | x) = px(1 − p)1 − x

ML Connection:
  • Maximum Likelihood Estimation (MLE)
  • Loss function = − log likelihood

Expectation

Expectation là giá trị trung bình kỳ vọng của một biến ngẫu nhiên.

E[X] = Σ x P(X=x)   (discrete)

E[X] = ∫ x f(x) dx   (continuous)

ML Connection:
  • Expected loss
  • Risk minimization

Variance

Variance đo độ phân tán quanh kỳ vọng.

Var(X) = E[(X − E[X])²]

ML Connection:
  • Bias–Variance tradeoff
  • Model stability
